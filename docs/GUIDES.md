# Guides

Practical usage guides for common `xfoil` workflows.

| Guide | Use it when |
| --- | --- |
| [Convergence](guides/convergence.md) | A point or sweep drops rows, stalls, or needs more careful operating-point setup. |
| [Custom Airfoils](guides/custom-airfoils.md) | You have coordinates from CAD, UIUC, a solver, or a generated shape. |
| [Bring Your Own Binary](guides/byo-binary.md) | You need to run before every `@xfoil/<platform>` package is published, or you want a custom XFOIL build. |
| [Browser Geometry](guides/browser-geometry.md) | You want to use airfoil generation, parsing, or display code in client-side apps without bundling Node APIs. |
