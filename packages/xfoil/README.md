# xfoil

Run headless XFOIL from TypeScript.

This package is the MIT-licensed wrapper. Platform executables are distributed
as separate GPL-licensed `@xfoil/<platform>` optional packages. See the
repository README and `docs/` directory for architecture, licensing, and binary
distribution details.

Run `npx xfoil --doctor` to verify binary resolution, print the XFOIL version,
and execute a small headless smoke test.
